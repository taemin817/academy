데이터타입과 변수

  • 숫자타입 : 1, 3.14, 70
  • 논리타입 : true, false
  • 문자열타입 : "hello", "a"
  • 객체타입

문자, 문자열. 자바스크립트는 문자열로만 취급함

문자열 "hello" : 문자(h e l l o)의 집합 문자 h : 자바스크립트는 문자가 없다.

변수

: 장소이지만 담는 것은 중요한, 내용(수). 값을 ('사용'하기위해) 저장하는 메모리 블럭(기억장소), 값을 기억하고 '사용'

변수선언 : 프로그래머가 메모리를 사용하는 방식 중의 하나

변수선언 방식

let, const(상수화변수)
